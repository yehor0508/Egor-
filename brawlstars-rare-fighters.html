Редкие бойцы


  • Барли — редкий боец класса «Разрушитель» (ранее «Метатель»). Его можно получить из любого ящика или при подключении к Supercell ID вместе со скином «Волшебник Барли».

  • Роза — редкий боец класса «Танк» (ранее «Тяжеловес»). Её можно получить из любого ящика.

  • Поко — редкий боец класса «Поддержка». Его можно получить из любого ящика.

  • Эль Примо — редкий боец, предназначенный для ближнего боя, класса «Танк» (ранее «Тяжеловес»). Его можно получить из любого ящика.
