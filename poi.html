Легендарные бойцы

  • Леон — легендарный боец класса «Убийца» (раньше «Скрытный убийца»). Его можно получить из ящиков, а также он довольно редко продаётся в магазине.

  • Спайк — легендарный боец класса «Разрушитель» (ранее «Стрелок»). Можно получить из любого ящика.

  • Сэнди — легендарный боец класса «Поддержка». Его можно получить из любого ящика.

  • Ворон — легендарный боец класса «Убийца» (ранее «Убийца с ядом»). Его можно получить из любого ящика.

  • Мэг — легендарный боец класса «Разрушитель». Её можно получить из любого ящика.

  • Ева — хроматический боец класса «Разрушитель». Её можно получить в 11 сезоне Brawl Pass в ящиках (после прохождения Brawl Pass до 30 уровня) и в премиальном Brawl Pass 11 сезона «Биокупол».
